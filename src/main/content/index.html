Runtimes & Languages

Use the right language for the job. Java is at the core of most modern application environments and can support open micro-profile or Spring-based microservices. Java is also for WebSphere Traditional applications that you have lifted and shifted into containers and wrapped with APIs. Node and Swift are for channels to various client devices.

Develop

Kabanero provides plugins for leading integrated development environments, which allows developers to extend their development environment to support inner loop development and testing. It also provides a hosted environment based on Eclipse Che.

Build

DevOps is critical for a successful microservice based deployment. Kabanero brings an open source CI/CD infrastructure based on the open source project Tekton, which allows for a rich set of stages that pipelines can use to integrate features like patching, vulnerability scanning, and more.

Manage

The application is deployed to Kubernetes, where it can be scaled, secured, and run at high throughput. Kabanero provides the ability to augment the base containers with additional management capabilities.
